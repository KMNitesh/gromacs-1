Welcome to the GitHub Page of my Gromacs fork.

The master branch in this fork implements Gaussian charges (instead of point-like ones). This project is now in pre-alpha stage: as soon as a bug-free version will be reached, we will officially release the software adding full documentation and examples. A publication in a peer-reviewed journal shall also follow. Till then, if you use the code, please let us know directly by sending us an email.

Thank you.
